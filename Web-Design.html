Introduction

My web-design journey first began in 2021 at the University of Huddersfield. I had briefly touched upon web-design in my high school IT classes but was never really captivated by the subject till I covered it at university. I soon realised that I had a passion for the subject as well as realising that I was quite proficient in the subject after scoring an A overall in the module. My first paid venture was with Robinson Fencing, where I did a complete overhaul of their website which you can read more about below.

The vast majority of my websites are designed using platforms like WordPress, all the websites you see below were in fact designed using WordPress.org and WordPress.com however, this website was written up in Sublime using HTML, CSS and JavaScript. It is hosted on GitHub as the site has no back-end, I made the decision to write this site up myself in order to have a more unique appearance and to have full reign of the design. Any websites that I do for clients are designed on WordPress however, as it is easier for the client to add content to their site at a later date as the client does not need knowledge on HTML, CSS or anyother mark up / programming language in order to add content to their site.

I have also designed websites on other platforms outside of WordPress including: Seriff Web Plus, Wix and Webs.
